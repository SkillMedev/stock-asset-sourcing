# Stock & Visual Asset Sourcing

**Find, license, fetch, and curate stock photos that don't look like stock.** — built in-house by [Skill&nbsp;Me](https://skillme.dev).

Everything from "I need an image of X" to a cohesive, legally-clear set on the page. Pick the right library (free, curated, paid, or editorial) and craft the search; confirm the license and whether you need a model or property release; pull images programmatically through the Unsplash, Pexels, and Pixabay APIs with their attribution and hotlinking rules handled; and art-direct the result so it reads as an intentional brand, not a stock dump. Use when you need real photography for a hero, blog, ad, deck, social post, or product, and want to source it well, ship it legally, and make it look considered.

⭐ **If this is useful, star the repo** — it's how we gauge what to build next.

## Install

- **From the catalog:** [skillme.dev/pack/stock-asset-sourcing](https://skillme.dev/pack/stock-asset-sourcing) — install the whole pack into Claude in one step.
- **With the skills CLI:** `npx skills add SkillMedev/stock-asset-sourcing`
- **Manually:** copy any `skills/<slug>/SKILL.md` into your Claude skills directory.

## Skills in this pack

- **[Find the Right Stock Photo](skills/stock-photo-finder/SKILL.md)** — Source the right stock photograph for a brief by picking the best library and crafting the search.
- **[Image License & Usage Rights](skills/image-license-rights/SKILL.md)** — Decide whether you can legally use a stock image, what the license requires, and whether you need a model or property release.
- **[Stock Photo APIs](skills/stock-photo-api/SKILL.md)** — Fetch stock photos programmatically through the Unsplash, Pexels, and Pixabay APIs, with the auth, rate-limit, attribution, and hotlinking rules each one requires.
- **[Visual Asset Curation](skills/visual-asset-curation/SKILL.md)** — Art-direct a set of stock images so they look intentional and on-brand instead of like stock, and choose and sequence the final picks.
